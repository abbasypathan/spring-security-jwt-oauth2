* Spring Security

- To enable the security we just have to add security dependency, It will ask for the username and password, username will be user & password is printed in console randomly generated

<dependency>
	<groupId>org.springframework.boot</groupId>
	<artifactId>spring-boot-starter-security</artifactId>
</dependency>

- We can configured our own username & password

spring.security.user.name = ${SECURITY_USER_NAME:admin}
spring.security.user.password = ${SECURITY_USER_PASSWORD:admin}
